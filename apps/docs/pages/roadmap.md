# Roadmap (High-level)

This is a **high-level** roadmap intended to prevent MVP documentation/marketing from implying features that are not fully shipped.

> This roadmap is not a promise or timeline.

## Planned

- **Real billing + payments** (replace mock upgrade flows with a real payment provider)
- **Adaptive quizzes** (targeted practice based on performance; deeper gap analysis)
- **Richer knowledge graph** (concept-level nodes and relationships beyond course/module/lesson)
- **Managed API keys** (server-hosted keys; not available in this build)
- **Marketplace trust signals** (real ratings/reviews + anti-spam), and monetization
- **Deeper analytics** (durable metrics, dashboards, export)
